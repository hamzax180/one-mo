## 🔥 About

ONEMO is a premium brand brand and full-service garment manufacturing company rooted in Istanbul, Türkiye. This website showcases the brand's identity, gallery, manufacturing capabilities, and provides customer support through an AI-powered chatbot.

**🌐 Live:** [one-mo.vercel.app](https://one-mo.vercel.app)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Dark Luxury Design** | Premium aesthetic with gold accents, custom cursor, and smooth animations |
| 🖼️ **Full-Bleed Hero** | Immersive hero section with parallax-style imagery |
| 📸 **Gallery** | Curated lookbook with lightbox overlay and hover effects |
| 🤖 **AI Support Chatbot** | Intelligent chatbot handling FAQs, sizing, shipping, returns & more |
| 📱 **Fully Responsive** | Seamless experience across desktop, tablet, and mobile |
| ✉️ **Contact Form** | Integrated with FormSubmit for real email delivery |
| 🎭 **Scroll Animations** | Reveal-on-scroll effects using Intersection Observer |
| 🔤 **Custom Typography** | Playfair Display + Inter from Google Fonts |
| ⚡ **Custom Form Validation** | Styled inline alerts replacing browser defaults |
| 🏭 **Manufacturing Showcase** | Services, capabilities, and brand development info |

## 🛠️ Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Google Fonts** — Playfair Display & Inter
- **Font Awesome 6** — Icon library
- **FormSubmit** — Backend-free form handling
- **Vercel** — Deployment & hosting

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/hamzax180/one-mo.git

# Open in browser
cd one-mo
start index.html
```

No build step required — it's pure HTML, CSS & JS.
